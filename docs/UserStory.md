# 유저 스토리

## 1️⃣ Member (회원) 도메인

### 인증 관련

```
US-M01: 회원가입
As a 신규 사용자
I want 이메일, 비밀번호, 이름, 연락처로 회원가입하고 싶다
So that 서비스에서 내 계정을 생성하고 주문과 배송 정보를 관리할 수 있다

US-M02: 로그인
As a 기존 회원
I want 이메일과 비밀번호로 로그인하고 싶다
So that 내 계정에 접속해 쇼핑 서비스를 이용할 수 있다

US-M03: 이메일 인증
As a 신규 회원
I want 가입 시 이메일 인증 링크를 받고 인증하고 싶다
So that 내 이메일 소유권을 확인하고 계정을 활성화할 수 있다
```

### 회원 정보 관리

```
US-M04: 내 정보 조회
As a 회원
I want 내 이름, 연락처 등 회원 정보를 조회하고 싶다
So that 현재 등록된 내 계정 정보를 확인할 수 있다

US-M05: 회원 정보 수정
As a 회원
I want 내 이름, 연락처, 비밀번호를 수정하고 싶다
So that 내 계정 정보를 최신 상태로 유지할 수 있다

US-M06: 회원 탈퇴
As a 회원
I want 서비스를 탈퇴하고 싶다
So that 더 이상 내 계정을 사용하지 않을 때 이용을 종료할 수 있다
```

### 배송지 관리

```
US-M07: 배송지 추가
As a 회원
I want 여러 배송지를 저장하고 싶다
So that 주문 시 배송 정보를 매번 새로 입력하지 않아도 된다

US-M08: 기본 배송지 설정
As a 회원
I want 여러 배송지 중 하나를 기본 배송지로 지정하고 싶다
So that 주문할 때 자주 사용하는 배송지가 자동으로 선택될 수 있다

US-M09: 배송지 수정/삭제
As a 회원
I want 저장된 배송지를 수정하거나 삭제하고 싶다
So that 배송지 정보를 최신 상태로 유지할 수 있다
```

### 예치금 관리

```
US-M10: 예치금 잔액 조회
As a 회원
I want 내 예치금 잔액을 조회하고 싶다
So that 주문 또는 결제 시 사용할 수 있는 금액을 확인할 수 있다
```

### 판매자 등록

```
US-M11: 판매자 등록 신청
As a 회원
I want 사업자 정보와 정산 계좌 정보를 등록해 판매자로 신청하고 싶다
So that 상품을 판매할 수 있는 권한을 요청할 수 있다

US-M12: 판매자 신청 상태 조회
As a 판매자 신청 회원
I want 내 판매자 등록 상태를 확인하고 싶다
So that 승인 대기, 승인 완료, 정지 여부를 확인할 수 있다
```

## 2️⃣ Product (상품) 도메인

### 상품 관리 (판매자 관점)

```
US-P01: 상품 등록
As a 승인된 판매자
I want ISBN, 제목, 저자, 출판사, 가격, 재고 수량, 도서 유형, 카테고리 정보를 입력해 상품을 등록하고 싶다
So that 내 상품을 플랫폼에서 판매할 수 있다

US-P02: 상품 등록 상태 확인
As a 판매자
I want 내가 등록한 상품의 검수 상태를 확인하고 싶다
So that 현재 판매 가능한 상태인지 알 수 있다

US-P03: 상품 정보 수정
As a 판매자
I want 내가 등록한 상품의 가격, 재고, 카테고리 등 판매 정보를 수정하고 싶다
So that 상품 정보를 최신 상태로 유지할 수 있다

US-P04: 재고 관리
As a 판매자
I want 상품의 재고 수량과 판매 가능 상태를 관리하고 싶다
So that 품절이나 잘못된 판매를 방지할 수 있다
```

### 상품 조회 (구매자 관점)

```
US-P05: 도서 검색
As a 구매자
I want ISBN, 저자, 제목으로 도서를 검색하고 싶다
So that 원하는 책을 빠르게 찾을 수 있다

US-P06: 카테고리별 조회
As a 구매자
I want 카테고리별로 도서를 조회하고 싶다
So that 관심 있는 분야의 상품을 쉽게 탐색할 수 있다

US-P07: 상품 상세정보 조회
As a 구매자
I want 도서의 기본 정보, 가격, 재고 상태, 평균 평점, 리뷰 수를 확인하고 싶다
So that 구매 여부를 판단할 수 있다
```

### 리뷰 (구매자 관점)

```
US-P08: 별점 등록
As a 상품을 구매한 회원
I want 구매한 상품에 별점을 남기고 싶다
So that 내 만족도를 다른 구매자들에게 전달할 수 있다

US-P09: 별점 조회
As a 구매자
I want 상품의 평균 별점과 별점 수를 확인하고 싶다
So that 다른 구매자들의 평가를 참고해 구매를 결정할 수 있다
```

## 3️⃣ Order (주문) 도메인

### 장바구니

```
US-O01: 장바구니에 상품 담기
As a 구매자
I want 관심 있는 상품을 장바구니에 담고 싶다
So that 나중에 여러 상품을 한 번에 주문할 수 있다

US-O02: 장바구니 수량 변경
As a 구매자
I want 장바구니에 담긴 상품의 수량을 변경하고 싶다
So that 원하는 수량만큼 주문할 수 있다

US-O03: 장바구니에서 상품 삭제
As a 구매자
I want 장바구니에서 불필요한 상품을 삭제하고 싶다
So that 주문 전에 상품 목록을 정리할 수 있다
```

### 주문 생성

```
US-O04: 주문 생성
As a 구매자
I want 장바구니 상품 또는 바로구매 상품으로 주문을 생성하고 싶다
So that 결제 단계로 진행할 수 있다

US-O05: 배송지 선택
As a 구매자
I want 등록된 배송지 중 하나를 선택해 주문에 반영하고 싶다
So that 상품이 올바른 주소로 배송될 수 있다

US-O06: 주문 결제 금액 확인
As a 구매자
I want 주문 시 예치금 사용액과 추가 결제 필요 금액을 확인하고 싶다
So that 예치금이 먼저 차감되고 부족한 금액만 별도로 결제할 수 있다
```

### 주문 관리

```
US-O07: 주문 조회
As a 구매자
I want 내 주문 목록과 주문 상태를 조회하고 싶다
So that 주문이 정상적으로 처리되고 있는지 확인할 수 있다

US-O08: 주문 취소
As a 구매자
I want 배송 시작 전에 주문을 취소하고 싶다
So that 결제 취소 및 환불 절차를 진행할 수 있다

US-O09: 배송 상태 조회
As a 구매자
I want 주문의 배송 상태와 운송장 번호를 확인하고 싶다
So that 상품이 언제 도착하는지 알 수 있다
```

## 4️⃣ Payment (결제) 도메인

### 결제

```
US-PA01: 주문 결제
As a 구매자
I want 주문 금액에 대해 예치금이 먼저 적용되고 부족한 금액만 토스 결제로 결제하고 싶다
So that 보유한 예치금을 우선 사용하면서 남은 금액만 추가 결제할 수 있다

US-PA02: 결제 결과 확인
As a 구매자
I want 내 주문의 결제 완료 여부와 결제 상태를 확인하고 싶다
So that 주문이 정상적으로 접수되었는지 알 수 있다

US-PA03: 결제 실패 재시도
As a 구매자
I want 결제에 실패했을 때 다시 시도하고 싶다
So that 주문을 다시 생성하지 않고 구매를 완료할 수 있다
```

### 환불

```
US-PA04: 환불 처리
As a 구매자
I want 취소된 주문에 대해 환불을 받고 싶다
So that 내가 결제한 금액을 돌려받을 수 있다

US-PA05: 환불 상태 조회
As a 구매자
I want 환불이 진행 중인지 완료되었는지 확인하고 싶다
So that 환불 처리 결과를 알 수 있다
```

### 예치금

```
US-PA06: 예치금 충전
As a 회원
I want 토스 결제를 통해 예치금을 충전하고 싶다
So that 이후 주문 결제 시 예치금을 사용할 수 있다

US-PA07: 예치금 내역 조회
As a 회원
I want 예치금의 충전, 사용, 환불 내역을 확인하고 싶다
So that 예치금 흐름을 파악할 수 있다
```

### 정산 (판매자 관점)

```
US-PA08: 정산 내역 조회
As a 판매자
I want 내 판매 건의 정산 내역과 정산 예정 금액을 조회하고 싶다
So that 수익 현황을 확인할 수 있다

US-PA09: 출금 신청
As a 판매자
I want 정산 가능한 금액에 대해 출금을 신청하고 싶다
So that 정산금을 내 계좌로 받을 수 있다

US-PA10: 정산 완료 확인
As a 판매자
I want 정산이 완료되었는지 확인하고 싶다
So that 실제 입금 여부를 알 수 있다
```

### 운영 / 정책 관리

```
US-PA11: 수수료 정책 관리
As a 관리자
I want 정산에 적용할 수수료 정책을 등록하고 조회하고 싶다
So that 서비스 운영 기준에 맞게 정산 금액을 계산할 수 있다
```
